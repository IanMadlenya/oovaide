CLangView Help
Last updated: Aug 31, 2015

Summary

The CLangViewer will display the AST as a tree where the nodes in the tree can be expanded. In addition to the standard nodes, CLang provides an interface that can display other related information for a given AST node. For example, a function such as getCursorSemanticParent can be selected to display the semantic parent of the selected AST node.

Command Line

The command line arguments are passed directly to the CLang parser. This means that standard compile arguments such as -I to define an include directory that is passed to the CLangView program.

Synopsis: 

        CLangView [args] [filename]
In addition to the compile command line arguments, the -p argument can be used to specify an Oovaide project directory. An example is: 
        CLangView -p/test/test-oovaide testSource.cpp

Find

Performing a Find will search the AST for the specified text.

Add (Plus)

An add can be performed to follow a related type.
