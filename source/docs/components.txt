
Component types
	- are currently stored in oovcde-comps.txt.
	- Components
		- read with ComponentsFile::getComponentNames in ComponentFinder.h
	- Comp-type-*
		- read with ComponentsFile::getComponentType
	- Components-init-ext-roots
		- read with ComponentsFile::getExternalRootPaths.

Include directory dependencies
	- are stored in oovcde-incdeps.txt.
		- read with IncDirDependencyMap in CppParser.h
	- are created during analysis phase

Library dependencies
	- Clump sorting
		- LibSym-*-dep.txt
		- ObjSymbols::getSortedLibFileNames
